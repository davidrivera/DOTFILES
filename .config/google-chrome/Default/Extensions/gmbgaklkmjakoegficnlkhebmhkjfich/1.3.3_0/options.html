Report an Issue

If you run into issues with this extension, please report them so they can be fixed.

  1. Ensure you’re running the latest version of the extension.
    • Copy and paste this into the omnibox: chrome://extensions
    • Click on the ☑ Developer Mode check box
    • Click on the Update Extensions Now button.
    If there is a new version available, the extension will now be updated. Has the issue been fixed in this version? If so, yay!
  2. See if the problem has already been reported by another person. A complete list and status of all pending issues is available at github.com/manastungare/google-calendar-crx/issues.
  3. If you have the same problem as one reported there, please copy all the text in the box below, and paste it as a comment under that specific issue.
  4. If nobody has reported a similar problem yet, then create a new issue, include all the information from the box below, and provide detailed steps on how to reproduce the problem.

Thank you for helping make this extension better!
